Tags i've learned

Tags:

Emphasis
a string of emphasized, slightly more important text. Screen readers will change their voice for this text.
Superscript text
defines text as being superscript.
Citation
a citation for another source, often used with quotations. A person's name, a URL, a book, a movie title, etc.
Highlighting text
used to highlight a piece of text for reference. The keywords in a search results page, the current navigation item, etc.
Horizontal line
represents a thematic break in the content. For example, a scene change or topic change.
